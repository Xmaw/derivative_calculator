Derivative Calculator

INFO
This is a program used to calculate the derivative of a given function.
The character to represent the variables should be "x" (Lower case x).
This script is not intended to support multiple different variables.

In this build the user will get prompted in the console to enter
the desired function on which the program is going to calculate the derivative.

AN EXAMPLE:
    Enter an input to calculate the derivative of: x^2
    The derivative of x^2 is 2x.
